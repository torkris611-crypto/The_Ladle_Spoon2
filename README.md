# The Ladle Spoon - Столовая

Веб-приложение для столовой Антона. Позволяет сотрудникам просматривать меню, выбирать бизнес-ланчи, оформлять заказы и оставлять отзывы.

## Возможности

### Для посетителей:
- Просмотр меню на сегодня
- Популярные блюда
- Бизнес-ланчи
- Корзина заказов
- Отзывы посетителей

### Для зарегистрированных пользователей:
- Оформление заказов
- История заказов
- Возможность оставлять отзывы
- Напоминания о выдаче

### Для администратора:
- Загрузка нового меню
- Управление пользователями
- Отметка выданных заказов
- Черный список

## Технологии

- **HTML5** - структура страниц
- **CSS3** - стили и анимации
- **JavaScript** - логика приложения
- **JSON** - хранение данных
- **Font Awesome** - иконки
